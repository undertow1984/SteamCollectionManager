Help & Controls

Controller Button Mappings

A (South)Activate / Select focused item. Opens game options on cards. Confirms selections in modals & dropdowns. Opens virtual keyboard on text fields.
B (East)Cancel / Back / Close current modal, dropdown, sidecar, filters, or virtual keyboard. Exits multi-select mode. Fallback: clears active search or selections.
X (West)Toggle focus between main game grid and top header controls (search box, folder dropdown, view buttons). Press again to return to grid.
Y (North)Toggle the full Filters & Sort panel open or closed (and focus it when opening).
LB (Left Bumper)Open/focus left sidebar (folder navigation). From sidebar, returns focus to main grid.
RB (Right Bumper)Open/focus right sidecar (Quick Move / Manage panel). From sidecar, returns focus to main grid.
LT (Left Trigger)Modifier for multi-select (used together with A for shift-like selection behavior). Double-tap + hold LT (3s) toggles multi-select mode (disables and clears selection if already active).
Back / SelectInstantly open the Change Folder dialog.
LS (Click) + RTGlobal shortcut (if enabled): Hold for 2 seconds when minimized to restore app. Press in background to bring to front. Press in foreground to toggle fullscreen.
Left Stick / D-PadNavigate focus up/down/left/right across cards, folders, buttons, and panels.
Right StickSmooth scroll the active view (main grid, sidecar, modal content, etc.).

Tip: When a text field is focused, A will open the on-screen virtual keyboard for controller text entry.

Extra Settings (Settings Modal)

Steam Web API Key — Required to fetch your owned games list from Steam. You can generate one for free at steamcommunity.com/dev. The key is stored locally and never sent anywhere except to Steam's official API.

Steam ID64 — Your 17-digit Steam ID (usually starts with 76561198...). You can look it up at sites like steamid.io or by viewing your profile URL.

Web Version Port — The TCP port used when running the web server directly (e.g. node server.mjs or npm start). The web UI will be available at http://localhost:PORT. Default: 3000.

Electron Version Port — Internal port used by the desktop Electron app. Changing this requires restarting the application. Default: 3001.

Allow games in multiple folders — When turned on, the same game can belong to several normal (static) folders at the same time — they behave more like tags. When turned off (the default), moving a game to a different folder removes it from the previous one. Dynamic (rule-based) folders are always allowed to contain the same game as other folders regardless of this setting.

Cached Data section — Shows status and "Refresh" / "Clear" buttons for background data (HLTB playtimes, Metacritic scores, reviews, media, licenses, etc.). Use these when you want to force an update without a full library refresh.
