# SteamCollectionManager
A utility to manage large steam libraries.  Front End Launcher. Collections Manager.

Installation
git clone https://github.com/undertow1984/SteamCollectionManager.git
cd SteamCollectionManager
npm install

Start Application as Web App
Navigate to folder and run
npm start
open browser and navigate to http://localhost:3000

Start Application as executable
Navigate to folder and run
npm run electron

Build Installer / Exe
Navigate to folder and run
npm run build
You should now see a dist folder which will have a standalone windows binary under win_extract and a windows installer
